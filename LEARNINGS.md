# Learnings

## What I Built
A Bank Account Management System with JUnit 5 unit testing.

## What I Learned
1. JUnit 5 - @Test, @BeforeEach annotations
2. Assertions - assertEquals, assertThrows
3. Custom Exceptions - InsufficientFundsException
4. Transaction History using ArrayList
5. Maven dependency management using pom.xml
6. TDD - Writing tests before/alongside code

## Hardest Part
Understanding assertThrows for testing exceptions.

## How I Fixed It
Learned that assertThrows takes the exception class
and a lambda function that triggers the exception.

## Key Takeaway
Every method needs both positive and negative tests.
Testing is what separates junior from senior developers.
